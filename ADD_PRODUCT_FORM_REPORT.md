   - Placeholder: "أدخل اسم المنتج"
   - التحقق: مطلوب للمتابعة

2. **السعر** (مطلوب)
   - نوع: Number Input
   - Placeholder: "أدخل السعر"
   - الوحدة: د.ا (دينار)
   - التحقق: مطلوب للمتابعة

3. **الوصف** (اختياري)
   - نوع: Textarea
   - الارتفاع: 112px (min-h-28)
   - Placeholder: "أدخل وصف المنتج"

4. **الوزن** (اختياري)
   - نوع: Number Input
   - الوحدة: كجم
   - Placeholder: "أدخل الوزن"

**التصميم:**
- خلفية متدرجة (Rose → Pink → Orange)

### ✅ الخطوة 2: التفاصيل

**الحقول:**

1. **القسم** (مطلوب)
   - نوع: Select Dropdown
   - المصدر: من قاعدة البيانات (trpc.categories.list)
   - يعرض: جميع الفئات المتاحة
   - التحقق: مطلوب للمتابعة

2. **كود المستودع** (مطلوب)
   - نوع: Select Dropdown
   - المصدر: من قاعدة البيانات (trpc.warehouses.list)
   - يعرض: كود المستودع
   - التحقق: مطلوب للمتابعة

3. **الألوان** (مطلوب)
   - نوع: Text Input (نصي)
   - صيغة: "أحمر، أزرق، أخضر"
   - يتم تقسيمها بفواصل
   - التحقق: يجب اختيار لون واحد على الأقل

4. **المقاسات** (اختياري)
   - نوع: Text Input (نصي)
   - صيغة: "S، M، L، XL"
   - يتم تقسيمها بفواصل
   - الخيارات المعرفة: S, M, L, XL

**التصميم:**
- نفس نمط الخطوة 1
- Select Dropdowns بارتفاع 48px-64px
- تصميم متجاوب

### ✅ الخطوة 3: الصور والفيديوهات

**الحقول:**

1. **الصور** (مطلوب)
   - نوع: File Upload (متعدد)
   - الحد الأقصى: 5 صور
   - القبول: جميع صيغ الصور (image/*)
   - العرض: شبكة 2-3 أعمدة
   - حجم المعاينة: 96px × 96px
   - إمكانية الحذف: زر X عند التمرير

2. **الفيديو** (اختياري)
   - نوع: File Upload (واحد)
   - القبول: جميع صيغ الفيديو (video/*)
   - العرض: مع عناصر تحكم التشغيل
   - حجم المعاينة: 128px ارتفاع
   - إمكانية الحذف: زر X

**ميزات الرفع:**
- منطقة Drag & Drop
- رسائل واضحة للمستخدم
- معاينة فورية للملفات
- أيقونات توضيحية (ImageIcon, Video)

   - التحقق من صحة الخطوة الحالية
   - تفعيل/تعطيل زر التالي
