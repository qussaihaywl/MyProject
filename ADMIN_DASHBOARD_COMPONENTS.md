## 2️⃣ المكونات الفرعية

### 📊 مكون KPI Card
**الوظيفة:** عرض مؤشر رئيسي مع أيقونة وتغيير النسبة

**الخصائص:**
```typescript
{
  icon: IconComponent,      // أيقونة lucide-react
  title: string,            // عنوان المؤشر
  value: string | number,   // القيمة
  change: number,           // نسبة التغيير
  color: 'gold' | 'pink' | 'rose'  // اللون
}
```

**الميزات:**
- ✅ تدرجات لونية ديناميكية
